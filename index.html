Simple Init

url and thead are required
When you use init function, use should set "url" and "thead" for second parameters
Which second parameters is object
Info :
When you use replace function, "url" and "thead" are not required
No matters what the result is, when you have an array data or object data
Example for array data :
0 : ["Sudrun", "XI", 90]
1 : ["Caridi", "X", 69]

Example for object data :
0 : { name : "Sudrun", class : "XI", grade : 90}
1 : { name : "Caridi", class : "X", grade : 69}

It'll automatically catch the data of your

Please to always be consistent in using JSON, either Array or Array of Object 
var arTable = arTable.init("#table", {
    url : "json.json",
    title: "Data Orang",
    thead: ["ID", "name", "Grade", "Passing", "Sex", "Birth", "Place"]
});

Objects Parameters

  • url [string]
    • Your url json.
    • Example: 
      url : "json/json.json"
  • title [string]
    • Give title for your table or you can leave it empty.
    • Example: 
      title : "Data Orang"
  • thead [array of string]
    • Your thead column name.
    • Example: 
      thead : ["ID", "Name", "Grade", "Passing", "Sex", "Birth", "Place"]
    • NOTE: if you use a table as container, it not required.
  • checkable [boolean]
    • Set it to true then the checkbox will showed (default is false).
    • Example: 
      checkable : true
  • numeric [boolean]
    • Set it to true then the numbers will showed (default is false).
    • Example: 
      numeric : true
  • scrollable [boolean]
    • Set it to false and the Scroll will be hide (default is true).
    • Example: 
      scrollable : true
  • height [integer]
    • NOTE : Be carefully if you set scrollable to false and you set the height of table.
    • Example: 
      height : 800
  • hide [array]
    • Example: 
      hide : [0, 1]
      OR 
      hide : ["id", "password"]

Action

The action and popup_action are objects that defines a clickable button or specifies the URL of the page the link goes to.
action have 5 keys :
  • text : name or text of button
  • href : your link
  • attr [Optional ] : Your optional attributes
  • params [Optional ] : custom parameter for your link (href) without adding a listener
    • data : get current data
    • return string
  • listener [Optional ] : your action for the listener and you can custom your link/href in it
    • data : get current data
    • events
    • index : get current index
    • request : request data to your server
    • object : object this
Example : 
arTable.init("#table-action", {
    url     : "json.json",
    title   : "Data Orang",
    thead   : ["ID", "Name", "Grade", "Passing", "Sex", "Birth", "Place"],
    action  : 
    [
        {
            text    : "Edit",
            cls     : "btn t1 blue",
            href    : "edit.php",
            params  : function (data)
            {
                var url     = this.href + "?id=" + data[0];
                return url;
            },
            listener: function  (data, event, index, req, element)
            {
                event.preventDefault();
                var url = element.getAttribute("href");
                req(url, function (result) {
                    if(result.error === null && arTable.isJSON(result.result))
                    {
                        // Success
                    }
                    else
                    {
                        // Error
                    }
                });
            }
        },
        {
            text    : "Delete",
            cls     : "btn t1 red",
            href    : "delete.php",
            params  : function (data)
            {
                var url     = this.href + "?id=" + data[0];
                return url;
            }
        }
    ]
});

Content Top & Bottom : Action

    • success
    • danger
    • warning
    • info
  • attr [Optional ] : Your optional attributes
  • params [Optional ] : custom parameter for your link (href) without adding a listener
    • data : get current data of the selected data
    • return string
  • listener [Optional ] : your action for the listener and you can custom your link/href in it

    • events

Content Top & Bottom : Form Filter

form_filter is an object that will search or filter data
form_filter have 5 keys :
  • text : name or text of form filter
  • for_key : key of Array or Array of Object [string OR integer]
  • type :
    • input
    • select
  • arr : only for select type
  • attr [Optional ] : Your optional attributes
NOTE: It take half of second (1/2 second) to change or update the data of table (to prevent fast changing)
Example : 
arTable.init("#table-filter", {
    url     : "json.php",
    title   : "Data Orang",
    thead   : ["ID", "Name", "Grade", "Passing", "Sex", "Birth", "Place"],
    content_top :
    {
        form_filter :  [
            { 
                text    : "Nama Siswa",
                type    : "input", 
                for_key : 1, 
                attr    : 'placeholder="Masukkan Nama ..."'
            },
            { 
                text    : "Kelulusan",
                type    : "select", 
                for_key : 3, 
                arr     : [
                    ["", "All"],
                    ["1", "Lulus"],
                    ["0", "Tidak Lulus"],
                ]
            }
        ]
    },
    content_bottom :
    {
        form_filter :  [
            { 
                text    : "Nama Siswa",
                type    : "input", 
                for     : 1, 
                attr    : 'placeholder="Masukkan Nama ..."'
            },
            { 
                text    : "Kelulusan",
                type    : "select", 
                for     : 3, 
                arr     : [
                    ["", "All"],
                    ["1", "Lulus"],
                    ["0", "Tidak Lulus"],

Insert Data

Method of insertData is for inserting data to table's internal caches of data.

NOTE:

Please to always be consistent in using JSON, either Array or Array of Object and length of array is same with your data before

Insert Array

If your table's internal caches of data is Array (also from init method to replace your table tag into Arjunane Table)
Example: 
var setArTable = arTable.init("#table-insert", {
    ...
});

document.getElementById("insert").addEventListener("click", function () { 
    var insert = [ 101, "Arjuna", 70, 1, "Male", "2020-05-10", "Pekajangan"];
    setArTable.insertData(insert);
});

Insert Array of Object

If your table's internal caches of data is Array of Object
Example: 
var setArTable = arTable.init("#table-insert", {
    ...
});

document.getElementById("insert").addEventListener("click", function () { 
    var insert = {
        "id"    : 101,
        "name"  : "Arjunane",
        "grade" : 70,
        "passing": 1,
        "birth" : "2020-05-10",
        "place" : "Pekajangan"
    };
    setArTable.insertData(insert);
});

Delete Data

Method of deleteData is for deleting data into table's internal caches of data.
deleteData take one parameter
  • key : integer or array of integer
Example: 
var deleteArTable = arTable.init("#table-delete", {
    ...
});

document.getElementById("delete").addEventListener("click", function () {

document.getElementById("delete").addEventListener("click", function () {

Example Deleting

    • success

  • text : string (Default is : "Info Alert")
  • button_text : string (Default is : "OK")

Show Confirm

The showConfirm method displays a dialog box with a specified message, along with an OK and a Cancel button.
A confirm box is often used if you want the user to verify or accept something.
showConfirm takes 3 arguments:
  • Object
    • type
      • warning
      • info (Default)
      • success
      • danger
    • title : string (Default is : "INFO")
    • text : string (Default is : "Info Confirm")
    • accept_text : string (Default is : "Yes")
    • abort_text : string (Default is : "No")

Filter

Sometimes you want to filter the data from the outside without use form_filter from content_top or content_bottom, then the filter method will useful for you.
The filter method takes 3 parameters:
  • value : value from your input or select option and anymore
  • index : set your index to filter for specific data from your index
    • integer OR string
    • Use "*" to find from all index
      WARNING: when you use '*' (star) to find all your data, it will replacing all your searches before
  • Searches type
    • io : IndexOf() [default]
    • eq : Equal ( === )
    • gt : Greater than ( > )
    • lt : Lower than ( < )
    • gte : Greater than Equal ( >= )
    • lte : Lower than Equal ( <= )

Example :

var filterArTable = arTable.init("#table-filter", {
    ...
});

document.getElementById("input-name").addEventListener("input", function () {
    var value        = this.value;
    var findForIndex = 1;
    filterArTable.filter(value, findForIndex);
});

document.getElementById("input-grade").addEventListener("input", function () {
    var value        = this.value;
    var findForIndex = 2;
    filterArTable.filter(value, findForIndex, "gte");
});

Req

The req method is used to request to your server
It takes 2 parameters :
  • url : string
  • callback : method
    This method takes 2 keys:
    • error
      • status
      • statusText
    • result

Exmaple:

arTable.req("your-url", function (res) {
    if(res.error !== null)
    {
        console.log(res.result);
        // success
    }
    else
    {
        console.log(res.error.status);
        console.log(res.error.statusText);
        // error
    }
});
